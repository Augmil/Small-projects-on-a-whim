Quick Method

将js常用功能封装为函数，便于快速调用，提升开发速度。

1.各类弹窗功能

1.1 警告框

类似于js自带的alrt()方法，可以弹出一个警告框并且强制用户确认才能退出。

使用方法如下：

Q.alert("需要显示的警告语")

1.2 输入框

向用户请求输入一些内容，而后执行回调函数，碍于js的限制，暂无法写出同步版本。

该函数有三个参数msg, defaultText, callback均为可选。msg为提示用户需要输入的内容，defaultText为默认的内容，callback为回调函数默认将输入的内容作为参数传入。

使用方法如下：


        Q.input('请输入一些内容后前往控制台查看','这是默认文本',console.log);

1.3 决策弹窗

让用户进行一些必要的决策或判断

该函数有两个参数text, callback均为可选。text为提示用户需要判断的内容，callback为回调函数，用户点击确定则传入true否则传入false。

使用方法如下：


            Q.confirm("你是否看到了该弹窗", console.log);

1.4 提示弹窗

提醒用户一些信息，用户能否看到并不重要。该弹窗不会影响用户操作并且不久后会自动消失，其存在的时间与现实的内容长度有关。

该函数仅有一个参数text该参数将作为提示信息显示出来。

使用方法如下：


            Q.tips("不必紧张这只是一个提示信息，几秒后就会自动消失。");

2. 常用函数

2.1 动态加载js脚本

利用js动态加载js文件，免于预先在html中引入的麻烦。只加载当前需要的，即用即拿，用完即删。

js内容如下：


 /**
 * 动态引入 js 文件
 */
function loadScript(src, callback) {
    let script = document.createElement('script');
    script.type = 'text/javascript';
    script.onload = () => {
        callback();
        script.remove();
        console.log('若引入的js无效，请回到此处删除 "script.remove();"');
    };
    script.src = src;
    document.getElementsByTagName('head')[0].appendChild(script);
    return script;
}

//调用方法
loadScript('js/loadScript.js', () => {
    console.log('文件 "js/loadScript.js" 加载成功');
});

//或者直接调用对象方法
Q.loadScript('js/loadScript.js', () => {
    console.log('文件 "js/loadScript.js" 加载成功');
});

2.1 动态加载css样式文件

通过引入css样式库，即时使用，避免无样式可用的尴尬

js内容如下：


 /**
 * 动态引入 css 文件
 */
function loadStyle(href, callback) {
    let link = document.createElement("link");
    link.rel = "stylesheet";
    link.type = "text/css";
    link.href = href;
    document.getElementsByTagName('head')[0].appendChild(link);
    link.onload = () => {
        callback();
    };
    return link
}

//调用方法
loadStyle('css/loadStyle.css', () => {
    console.log('文件 "css/loadStyle.css" 加载成功');
});

//或者直接调用对象方法
Q.loadStyle('css/loadStyle.css', () => {
    console.log('文件 "css/loadStyle.css" 加载成功');
});
